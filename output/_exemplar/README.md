# スライドの完成見本（4枚セット）

## これは何？

**slide-upa** というツールで作る「プレゼン用スライド」の**見本**です。  
PowerPoint のようなスライドを、ブラウザ（Chrome など）で見られる形にしたものです。

- 開き方: このフォルダの `audience.html` をブラウザで開く  
- 操作: キーボードの **→（右矢印）** で次のスライド、**←（左矢印）** で前のスライド

---

## 4種類のスライド

| 番号 | 名前 | 何のため？ |
|------|------|------------|
| ① | 表紙 | 発表のタイトルを大きく見せる |
| ② | 要点リスト | 箇条書きで「今日のゴール」などを並べる |
| ③ | 大きな一言 | 一番伝えたい言葉を、ウパ博士の吹き出しで強調する |
| ④ | 画像メイン | 図や AI で作った画像を大きく載せる |

---

## 今回どこを変えたか（やさしい説明）

### ① 表紙

- 背景を**薄い水色**にした
- うしろに**ウパ博士の薄いシルエット**（透かし）を入れた
- タイトルの文字色を**濃い青緑**にして、白い影をつけて読みやすくした
- 下にあった「ページ番号」と小さなアイコンは**非表示**にした

### ② 要点リスト

- 見出し（例: `1-1. 今日のゴール`）を**左上**に固定した
- 見出しの下に**薄い横線**を引いて、タイトルと本文を区切った

### ③ 大きな一言

- 見出しも **② と同じく左上**
- **ウパ博士を左**、**吹き出しを右** に配置した（漫画のセリフのような形）
- 吹き出しの中で、重要なフレーズだけ**大きく・色付き**で目立たせた

### ④ 画像メイン

- 見出しは **②③ と同じく左上**
- ウパ博士やパニっくんの**サンプル画像は入れない**ことにした  
  → ここには Cursor や他の AI で作った**図・イラスト**を後から入れる想定
- 画像がまだないときは、**点線の枠**だけが表示される（「ここに画像を入れる場所」の目印）

## ファイルの役割

| ファイル | 役割 |
|----------|------|
| `audience.html` | スライド4枚の中身 |
| `slides.css` | 色・文字サイズ・レイアウトのルール |
| `slides.js` | 矢印キーでページをめくる処理 |
| `images/` | スライドで使う画像（キャラクターなど） |

---

## ④ に画像を入れる方法（後から）

1. 画像ファイルを `images/` フォルダに入れる  
2. `audience.html` の ④ の部分に、次の1行を追加する:

```html
<img class="slide__visual-img" src="images/ファイル名.png" alt="">
```

画像を入れると、点線の枠は自動的に消えます。
