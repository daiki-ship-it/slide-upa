# スライドの完成見本（7種セット）

## これは何？

**slide-upa** で使える **7種類のスライド型** を、1枚ずつ並べた見本です。

- 開き方: このフォルダの `audience.html` をブラウザで開く
- 操作: キーボードの **→（右矢印）** で次、**←（左矢印）** で前

新しいプロジェクトを生成するとき、このフォルダの CSS / JS などがコピー元になります。

---

## 7種類のスライド（この順番）

| 枚 | 型 | 名前 | 何のため？ |
|----|-----|------|------------|
| 1 | ① 表紙 | `title` | 講義全体のタイトル |
| 2 | ⑤ 章 | `chapter` | 章の切り替わり（アジェンダと同系統。該当章の丸だけ塗る） |
| 3 | ⑥ ゴール | `goal` | 今日のゴール（淡いヘッダー + ①②③） |
| 4 | ⑦ アジェンダ | `agenda` | 今日の流れ（淡いヘッダー + 縦タイムライン） |
| 5 | ② 要点 | `bullets` | 説明用の箇条書き |
| 6 | ③ 一言 | `quote` | ウパ博士の吹き出しで強調 |
| 7 | ④ 画像 | `visual` | 図やキャプチャを載せる |

---

## 各型の見分け方

### ① 表紙

- 全面が薄い水色、中央に大きなタイトル
- うしろにウパ博士の透かし

### ⑤ 章

- 上部が薄い水色のヘッダー（`CHAPTER` + 章タイトル）
- 本文はアジェンダと同じ縦タイムライン。**いまの章の丸だけ塗る**
- 台本の `## 章タイトル` から生成（アジェンダ節の項目を転用）

### ⑥ ゴール

- 上部が薄い水色のヘッダー（`TODAY'S GOALS` + タイトル）
- 本文は ①②③ の角丸バッジ + ゴール文
- 台本の見出しに「ゴール」を含む `###` 節から生成

### ⑦ アジェンダ

- 上部が薄い水色のヘッダー（`TODAY'S AGENDA` + タイトル）
- 本文は縦線 + 丸印 + 章ラベル + 説明文。**丸はすべて塗る**
- 台本の見出しに「アジェンダ」を含む `###` 節から生成

### ② 要点

- 左上に見出し + 横線
- 箇条書きが縦に並ぶ

### ③ 一言

- 左上に見出し
- 左にウパ博士、右に吹き出し

### ④ 画像

- 左上に見出し
- 下に点線枠（画像未設定時）。studio からアップロード可

---

## ファイルの役割

| ファイル | 役割 |
|----------|------|
| `audience.html` | 7枚のスライド本体 |
| `deck.json` | 型・見出し・台本テキストのメタデータ |
| `script.md` | 台本サンプル（生成スキルの形式例） |
| `slides.css` | 色・レイアウトの正本 |
| `slides.js` | 矢印キーでページ送り |
| `images/` | キャラクター画像（生成時にコピー） |

---

## ④ に画像を入れる方法

1. 画像を `images/` に入れる
2. `audience.html` の ④ の枠内に次を追加:

```html
<img class="slide__visual-img" src="images/ファイル名.png" alt="">
```

点線の枠は自動的に消えます。
